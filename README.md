# Telegram bot using FastAPI

FastAPI Telegram bot app

### Setup
- Create venv: 
  ```shell
  python -m venv .venv
  ```
- Activate venv:
  - Linux/UNIX:
    ```shell
    source .venv/bin/activate
    ```
  - Windows:
    ```powershell
    .venv\Scripts\activate
    ```
- Install dependencies: 
  ```shell
  pip install -r requirements.txt
  ```

- Run server:
  ```shell
  python -m fastapi dev main.py
  ```
